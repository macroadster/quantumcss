Philosophy

How Aurora ships

Step 01

Start with HTML

Headings, paragraphs, buttons, and forms already look considered — no class required.

Step 02

Name the component

Add card, nav-header, or starlight-card when you need structure and surface.

Step 03

Utility only if needed

Reach for spacing, grid, or responsive attributes when the design diverges — not before.

Step 04

Ship one file

Link quantum.min.css. Optional starlight.js for stars and theme helpers.
